# Agent Orchestrator

An open-source control plane for deploying, running, and governing AI agents safely at scale.

## Why?
AI agents are easy to build but hard to manage.
This project brings structure, safety, and cost control to agent-based systems.

## Core Capabilities
- Agent runtime & execution
- Tool orchestration
- Memory management
- Cost tracking & governance
- Safety & policy enforcement
